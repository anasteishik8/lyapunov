# Проект: Модель "Жертва-Хищник" с управлением (метод Ляпунова)

## Аннотация

Данный проект представляет комплексное исследование системы "жертва-хищник" с синтезом стабилизирующего управления и формальным доказательством устойчивости методом Ляпунова. Работа выполнена в соответствии с современными требованиями к научным исследованиям в области теории управления и математического моделирования.

## 1. Введение

### Актуальность работы

Модели типа "жертва-хищник" широко применяются в различных областях науки и техники:

- **Экология**: управление биоресурсами, сохранение видов, контроль экосистем
- **Эпидемиология**: моделирование распространения заболеваний
- **Экономика**: анализ конкурентного взаимодействия на рынке
- **Социология**: динамика взаимодействующих социальных групп

Управление такими системами позволяет достичь желаемого поведения и стабилизировать ключевые показатели на заданных уровнях.

### Цель работы

Построить стабилизирующее управление для системы жертва-хищник, обеспечивающее стабилизацию популяции хищников на заданном уровне x₂\* с формальным доказательством устойчивости методом Ляпунова.

### Задачи исследования

1. Исследовать поведение системы без управления
2. Синтезировать управление по методике АКАР
3. Провести формальное доказательство устойчивости по Ляпунову
4. Выполнить численное моделирование и верификацию
5. Проанализировать качество различных стратегий управления

## 2. Теоретическая часть

### 2.1 Классическая модель Лотки-Вольтерра

Базовая система описывается дифференциальными уравнениями:

```
ẋ₁ = a₁x₁ - β₁x₁x₂  (динамика жертв)
ẋ₂ = -a₂x₂ + β₂x₁x₂  (динамика хищников)
```

где:

- `x₁(t)` - популяция жертв в момент времени t
- `x₂(t)` - популяция хищников в момент времени t
- `a₁ > 0` - коэффициент роста жертв
- `a₂ > 0` - коэффициент убыли хищников
- `β₁ > 0` - интенсивность хищничества
- `β₂ > 0` - эффективность преобразования жертв в хищников

### 2.2 Метод Эйлера для численного решения

Для численного интегрирования используется явная схема Эйлера:

```
x₁(k+1) = x₁(k) + dt·f₁(x₁(k), x₂(k))
x₂(k+1) = x₂(k) + dt·f₂(x₁(k), x₂(k))
```

где `dt` - шаг интегрирования.

### 2.3 Методика АКАР

Алгебраический критерий абсолютной устойчивости (АКАР) используется для синтеза стабилизирующего управления на основе анализа характеристического полинома линеаризованной системы.

## 3. Постановка задачи

### Рассматриваемая система

```
ẋ₁ = a₁x₁ - β₁x₁x₂ + u(x₁,x₂)  (управляемая динамика жертв)
ẋ₂ = -a₂x₂ + β₂x₁x₂            (динамика хищников)
```

### Управляющее воздействие

`u(x₁,x₂)` - управление по популяции жертв (например, искусственное разведение/изъятие)

### Цель управления

Стабилизировать популяцию хищников на заданном уровне `x₂*` при обеспечении устойчивости системы.

### Ограничения

- `x₁(t) ≥ 0, x₂(t) ≥ 0` (неотрицательность популяций)
- `|u(t)| ≤ u_max` (ограничение управления)
- Минимизация энергозатрат на управление

## 4. Дискретизация системы

Применяется метод Эйлера с шагом `dt = 0.01`:

```
x₁(k+1) = x₁(k) + 0.01·(a₁x₁(k) - β₁x₁(k)x₂(k) + u(k))
x₂(k+1) = x₂(k) + 0.01·(-a₂x₂(k) + β₂x₁(k)x₂(k))
```

### Анализ точек равновесия

Система без управления имеет две точки равновесия:

- **Тривиальная**: `(0, 0)` - вымирание обеих популяций
- **Нетривиальная**: `(a₂/β₂, a₁/β₁)` - сосуществование популяций

## 5. Синтез управления по АКАР

### Целевая точка стабилизации

Для заданного `x₂*` соответствующее равновесное значение:

```
x₁* = a₂/β₂
```

### Закон управления

Используется линейное управление вида:

```
u(x₁,x₂) = -T₁·ψ + T₂·(x₁ - φ(x₂))
```

где:

- `ψ = x₂ - x₂*` - отклонение от целевого значения
- `φ(x₂) = a₂/β₂` - функция настройки
- `T₁, T₂` - параметры управления

### Поиск оптимальных параметров

Параметры `T₁` и `T₂` выбираются из условия устойчивости линеаризованной системы по критерию Рауса-Гурвица.

## 6. Доказательство устойчивости методом Ляпунова

### Выбор функции Ляпунова

Используется квадратичная функция:

```
V(e₁, e₂) = e₁² + e₂²
```

где `e₁ = x₁ - x₁*`, `e₂ = x₂ - x₂*` - отклонения от равновесия.

### Условия устойчивости

1. **Положительная определенность**: `V(e₁, e₂) > 0` при `(e₁, e₂) ≠ (0, 0)`
2. **Отрицательность производной**: `dV/dt < 0` вдоль траекторий системы

### Анализ линеаризованной системы

Матрица Якоби в точке равновесия:

```
J = [[a₁ - β₁x₂* - T₂,  -β₁x₁*    ],
     [β₂x₂*,            -a₂ + β₂x₁* - T₁]]
```

Условия устойчивости по Раусу-Гурвицу:

- `tr(J) < 0` (след отрицателен)
- `det(J) > 0` (определитель положителен)

### Заключение об устойчивости

При выполнении условий устойчивости система с управлением асимптотически устойчива в окрестности точки равновесия.

## 7. Численное моделирование

### Тестовые сценарии

Проводится моделирование для различных начальных условий:

1. Отклонение вправо-вверх от равновесия
2. Отклонение влево-вверх от равновесия
3. Отклонение вправо-вниз от равновесия

### Показатели качества

- **Время установления**: время достижения 5% точности
- **Максимальное управление**: пиковое значение управляющего сигнала
- **Среднее управление**: среднее значение по модулю
- **Финальная ошибка**: отклонение от цели в конце моделирования

### Сравнение стратегий

Тестируются различные подходы:

- **Оптимальное линейное** - с найденными параметрами
- **Консервативное** - с уменьшенными коэффициентами
- **Агрессивное** - с увеличенными коэффициентами
- **Нелинейное** - адаптивное управление
